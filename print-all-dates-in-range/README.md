# print-all-dates-in-range
Print all dates in a range (inclusive). Specified via command line arguments. eg:


```
./print_dates.py 2011-01-01 2011-01-10
Sat January 01 2011
Sun January 02 2011
Mon January 03 2011
Tue January 04 2011
Wed January 05 2011
Thu January 06 2011
Fri January 07 2011
Sat January 08 2011
Sun January 09 2011
Mon January 10 2011
```
